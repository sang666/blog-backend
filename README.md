## san_blog
### 内容 
- 顾名思义，一个博客网站，对阳光沙滩博客用mybatis-plus和ElasticSearch以及阿里云oss进行改写
- [后端-已完成](https://github.com/sang666/ok_blog)
- [前端管理中心-已完成](https://github.com/sang666/blog-vue-admin)

## 技术栈
- 前后分离架构
### 后端
- springboot
- mybatis-plus
- ElasticSearch
- Nginx
- Redis
- spring security
- mysql
- jwt
- 阿里云oss

### 前端
- vue.js
- nuxt.js
- element-ui
- iview
