## san_blog
### 内容 
- 顾名思义，一个博客网站，对b站阳光沙滩博客用mybatis-plus和ElasticSearch以及阿里云oss进行改写
- [发布教程的up主(程序员拉大锯)的B站链接大家可以去关注下](https://space.bilibili.com/44272436)

## 技术栈
- 前后分离架构
### 后端
- springboot
- mybatis-plus
- ElasticSearch
- Nginx
- Redis
- spring security
- mysql
- jwt
- 阿里云oss

### 前端
- vue.js
- nuxt.js
- element-ui
- iview
